# 健康卫士-HealthGuard
### 基于MVP+Retrofit+RxJava+GreenDao+Dagger2+ButterKnife等开源架构实现的一个健康资讯类app,实现了健康知识，养生菜谱，药品速查等功能
***
### API数据源自[天狗云](http://www.tngou.net)
***
